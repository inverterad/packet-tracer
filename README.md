# Placeholder

Projektuppgift i kursen Nätverkssäkerhet via Frans Schartau

Uppgiften är att bygga upp en testmiljö för ett företagsnätverk med två siter på olika geografiska platser: Stockholm och Göteborg. Varje nätverk ska vara segmenterat i olika zoner för att kunna kontrollera trafikflöden och öka säkerheten. Kommunikation mellan de olika nätverken på respektive plats sker genom en ISP med hjälp av en virtuell tunnel.
